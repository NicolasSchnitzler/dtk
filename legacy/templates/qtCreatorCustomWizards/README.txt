# INSTALL A CUSTOM WIZARD FOR QtCREATOR
#
##

You just need to copy one of the current directory into the QtCreator
wizard directory which can be one of the following:

- share/qtcreator/templates/wizards
- the local user's configuration folder, $HOME/.config/Nokia/qtcreator/templates/wizards
- %APPDATA%\Nokia\qtcreator\templates\wizards

For more information refer to:

http://doc.qt.digia.com/qtcreator/creator-project-wizards.html#overview-of-custom-wizards
